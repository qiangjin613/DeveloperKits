# Maven功能与局限

Maven 是 Apache 软件基金会组织维护的一款专门为 Java 项目提供构建和以来管理支持的工具。

1. 作为依赖管理工具
   
   对Jar包的规模、来源、依赖关系进行管理。

2. 作为构建管理工具
   
   主要包含：清理、编译、测试、报告、打包（Java工程：JAR包、Web工程：WAR包）、安装、部署。
